# WindowTable

<div className="example">
  <WindowTable
    width={497}
    height={297}
    rowCount={100}
    rowHeight={35}
    columnCount={100}
    columnWidth={80}
    fixedLeftCount={1}
    fixedRightCount={1}
    fixedTopCount={1}
    fixedBottomCount={1} >
    {({ columnIndex, rowIndex, style }) => (
      <div className={(columnIndex + rowIndex) % 2 ? 'cell-even' : 'cell-odd'} style={style}>
        {rowIndex} , {columnIndex}
      </div>
    )}
  </WindowTable>
</div>

React table component that supports fixed columns inspired by [react-window](https://github.com/bvaughn/react-window).

[![NPM registry](https://img.shields.io/npm/v/react-window-table.svg?style=for-the-badge)](https://www.npmjs.com/package/react-window-table) ![NPM license](https://img.shields.io/badge/license-mit-red.svg?style=for-the-badge)
